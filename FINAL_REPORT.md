## 🚀 如何开始赚钱

### 第一步：部署项目（5分钟）

```bash
# 运行一键部署脚本
./auto-deploy.sh
```

脚本会自动完成：
- 初始化Git仓库
- 创建GitHub Pages工作流
- 提交所有文件
- 推送到GitHub

### 第二步：配置赞助链接（2分钟）

```bash
# 添加赞助链接
./add-sponsor-links.sh

# 输入你的GitHub用户名，例如：hsh-oc01
```

### 第三步：启用GitHub Pages（3分钟）

1. 访问你的GitHub仓库
2. Settings → Pages
3. Source选择：`Deploy from a branch`
4. Branch选择：`main`，文件夹：`/ (root)`
5. 保存

### 第四步：等待部署（1-2分钟）

访问：`https://你的用户名.github.io/ai-money-guide`
